# Sistem Informasi Pelayanan Surat Menyurat Berbasis Web di Kantor Kelurahan Anotaurei Kecamatan Anotaurei Kabupaten Kepulauan Yapen 

## Deskripsi
Sistem Informasi Surat Menyurat merupakan aplikasi berbasis web yang dirancang untuk mendukung proses administrasi pelayanan surat di Kantor Kelurahan Anotaurei Kecamatan Anotaurei.

Sistem ini memungkinkan masyarakat untuk melakukan pengajuan surat secara online serta membantu pihak kelurahan dalam melakukan verifikasi, pengelolaan data, hingga pencetakan surat secara efisien dan terstruktur.

## Tujuan
Tujuan pengembangan sistem ini adalah:
1. Meningkatkan kualitas pelayanan publik
2. Mempermudah proses pengajuan surat bagi masyarakat
3. Mengurangi penggunaan dokumen fisik
4. Mempercepat proses administrasi

## Fitur Sistem

### Pengguna (Masyarakat)
- Registrasi dan Login
- Pengajuan Surat
- Monitoring Status Pengajuan
- Unduh dan Cetak Surat

### Admin Kelurahan
- Verifikasi Pengajuan Surat
- Pengelolaan Data Penduduk
- Pengarsipan Surat
- Pembuatan Laporan

## Teknologi
- PHP Native
- Database MySQL
- HTML, CSS, JavaScript

## Implementasi
Sistem ini diimplementasikan menggunakan arsitektur berbasis web dengan model client-server.

## Penulis
Maya Melanesia Rumasukun
